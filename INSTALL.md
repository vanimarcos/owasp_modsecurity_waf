## Installation

1. Clone the repo and access its folder via terminal

2. Create a .env file in the current folder and setup the var
```sh 
touch .env 
```

2. Run the code in the terminal 

```sh 
chmod +x install.sh
```

3. Run the file install.sh with privileges

```sh
sudo ./install.sh
```
